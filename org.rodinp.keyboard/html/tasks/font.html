Changing the font

In order to display the Unicode characters for the mathematical language, there is a special font associated with the Text area. This font can be changed in the Prefereces Page associated with the Colors and Fonts. It can be opened from the category "General/Appearance/Colors And Fonts". Any font can be choosen, but mathematical language formulas can be displayed correctly only if the chosen font contains the Unicode characters for the mathematical language. From the Preferences page, you can choose the font, the style and the size for the Text area. (More information on Preferences Page is here.)
